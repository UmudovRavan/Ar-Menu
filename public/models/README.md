# 3D Models Qovluğu

Bu qovluqda yemək 3D modelləri yerləşdirilməlidir.

## Dəstəklənən Formatlar
- `.glb` (tövsiyə olunan)
- `.gltf`

## Model Tələbləri
- Ölçü: 5MB-dan az olmalıdır
- Polygons: 50k-dan az tövsiyə olunur
- Texture: 2048x2048 maksimum

## Mövcud Modellər
Aşağıdakı modelləri əlavə edin:

1. `burger.glb` - Burger 3D modeli
2. `pizza.glb` - Pizza 3D modeli  
3. `salad.glb` - Salat 3D modeli
4. `kebab.glb` - Kabab 3D modeli

## Pulsuz 3D Model Mənbələri
- [Sketchfab](https://sketchfab.com/search?q=food&type=models)
- [Poly Pizza](https://poly.pizza/)
- [TurboSquid Free](https://www.turbosquid.com/Search/3D-Models/free/food)
- [Free3D](https://free3d.com/3d-models/food)

## Qeyd
Əgər model tapılmasa, avtomatik olaraq fallback cube göstəriləcək.
